## 2. IC-7000 周波数表示ズレの確認

- LSBで−300Hz、USBで+300Hzのズレが発生
- **結論：仕様**（SSBの抑圧搬送波方式による）
- MMSSTVでの対応：USBダイヤルを−300Hz低く設定するか、そのままでもOK（SSTVは許容範囲内）

## 3. zerostack のトラブル対応

### ① 402エラー（クレジット不足）
- 原因：max_tokensが65536と大きすぎてクレジット不足
- 対応：設定ファイルを作成しmax_tokensを4096に変更

### ② 設定ファイルの作成
- 場所：`~/.config/zerostack/config.json`
- 最適設定：
  ```json
  {
    "provider": "openrouter",
    "model": "qwen/qwen3-coder:free",
    "max_tokens": 4096,
    "reserve_tokens": 1024,
    "keep_recent_tokens": 8000,
    "compact_enabled": true
  }
  ```

### ③ 429エラー（レート制限）
- 原因：無料モデルの混雑（Venice プロバイダー）
- 対応：別の無料モデルに切り替え

### ④ モデル自動選択スクリプト作成
- ファイル：`~/zerostack_model_select.sh`
- 動作：利用可能な無料モデルを自動検索してconfig.jsonを更新
- 使い方：zerostack起動前に実行

## 5. Firefox の Snap版 → apt版 への移行

### ① アップデート内容の確認
- `sudo apt full-upgrade --dry-run` で事前確認
- Snap版（`1:1snap1-0ubuntu5`）→ apt版（`151.0.1+build1-0ubuntu0.24.04.1~mt1`）へ切り替わることを確認
- 提供元：Mozilla Team PPA

### ② バックアップ
- Snap版プロファイル：`~/snap/firefox/common/.mozilla/firefox/`
- apt版プロファイル：`~/.mozilla/firefox/`
- バックアップコマンド：
  ```bash
  cp -r ~/.mozilla/firefox ~/.mozilla/firefox_backup_$(date +%Y%m%d)
  ```

### ③ アップデート実行
- `sudo apt full-upgrade` を実行
- apt版 Firefox 151.0.1 のインストール完了

### ④ Snap版の削除
- `sudo snap remove firefox` を実行
- about画面の表示が変化：
  - 変更前：`Mozilla Firefox Snap for Ubuntu`
  - 変更後：`Mozilla Firefox for Ubuntu` ✅

### ⑤ 最終確認
- `/usr/bin/firefox`（apt版）で正常動作
- Snap版は完全に削除済み
